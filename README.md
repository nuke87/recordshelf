# RecordShelf

RecordShelf is an open-source desktop app to catalog your **vinyl records** and **CDs**.
It stores your collection locally and can enrich entries by fetching **cover images**, **tracklists**, and metadata from public music databases.

## Features (planned)
- Local library for records and CDs (offline-first)
- Search, filter, and sorting
- Album details with cover + tracklist
- Enrichment via APIs:
  - Discogs (primary)
  - MusicBrainz (fallback)
- Manual edits and custom fields

## Tech
- **Go**
- **SQLite**
- Desktop UI: **Fyne** (default) or **Tauri** (optional)
